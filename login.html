Login Cracker v2 Simulator

Simulasi alat dynamic instrumentation untuk memantau dan mengalihkan request URL

Deskripsi Script

Script Frida ini melakukan hooking pada:

  • Konstruktor URL Java
  • Method HttpURLConnection.connect()
  • Method HttpURLConnection.getInputStream()
// Hook URL constructor
URL.$init.overload('java.lang.String').implementation = function (url) {
  // conditional redirect
  if (url.indexOf("prime.infernomods.fun/RVerify") !== -1) {
    var newUrl = url.replace("prime.infernomods.fun/RVerify", "YourWebsite.astralero/connect");
    console.log("[!] Redirected -> " + newUrl);
    return this.$init(newUrl);
  }
  return this.$init(url);
};

Script ini akan mengalihkan semua request yang menuju ke "prime.infernomods.fun/RVerify" ke server yang ditentukan pengguna.

Simulasi Redirect

URL asli yang akan diakses:

https://prime.infernomods.fun/RVerify?user=test&pass=123

Setelah di-intercept oleh script:

https://YourWebsite.astralero/connect?user=test&pass=123

Cara Penggunaan Frida

Untuk menggunakan script Frida asli:

  1. Install Frida pada perangkat dan komputer
  2. Simpan script sebagai hook.js
  3. Jalankan aplikasi target
  4. Eksekusi perintah: frida -U -f com.aplikasi.target -l hook.js
  5. Pantau output di konsol untuk melihat request yang di-intercept

Detil Teknis

Script ini menggunakan Frida untuk melakukan:

  • Runtime Hooking - Memodifikasi perilaku aplikasi saat dijalankan
  • URL Monitoring - Mencatat semua pembuatan objek URL
  • Conditional Redirect - Mengubah URL berdasarkan pola tertentu
  • Stack Tracing - Melacak asal panggilan URL untuk analisis
